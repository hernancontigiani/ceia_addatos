# Preparar el entorno
Preparar e instalar entorno en Linux
```
$ virtualenv <env_name>
$ source <env_name>/bin/activate
(<env_name>)$ pip install -r requirements.txt
```

Preparar e instalar entorno en Windows
```
$ python -m virtualenv <env_name>
$ .\<env_name>\Scripts\activate
(<env_name>)$ pip install -r .\requirements.txt
```
